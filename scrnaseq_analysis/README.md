# Single-cell RNA-seq Analysis

## Objective
Analyze gene expression across different cell types during Drosophila spermatogenesis using single-cell RNA-seq data from Witt et al. 2019.

## Background
Single-cell RNA-seq enables the identification of cell-type-specific transcriptional profiles and reveals heterogeneity within tissues.

## Methods
- Data normalization
- Clustering of cell types
- Gene expression analysis
- Visualization (heatmaps, UMAP, etc.)

## Tools
- R (Seurat / ComplexHeatmap)
- Python (optional)

## Focus
- Identification of cell-type-specific genes
- Analysis of post-meiotic gene expression
